# skills-hub

常用 skills 集合站，更多 skills 可在 [https://skills-hub.cc](https://skills-hub.cc) 了解。

## 技能
- `dev-deploy`：快速创建并部署 Web 应用到 Cloudflare Pages（可选 GitHub 推送）。

## 安装
将下面提示词交给你的 Agent（如 OpenClaw、Claude Code、Cursor）即可：

```text
请从 https://github.com/vibe-any/skills-hub 安装并启用 dev-deploy skill。安装完成后，简要确认已就绪，并提示我如何开始使用。
```
## 贡献
欢迎提交 PR，让你的独门技能普惠更多人。
