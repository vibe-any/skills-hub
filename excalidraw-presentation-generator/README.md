# Excalidraw 演示文稿生成器 Skill

这是一个基于实战经验总结的 Claude Skill，用于创建专业的 Excalidraw 演示文稿。

## 📁 文件结构

```
excalidraw-skill/
├── SKILL.md                          # Skill 定义（主文件）
├── README.md                         # 使用说明（本文件）
└── templates/                        # 模板文件夹
    ├── basic-presentation.json       # 基础演示模板
    ├── technical-tutorial.json       # 技术教程模板
    └── comparison-layout.json        # 对比布局模板
```

## 🚀 如何使用

### 自动触发

当你在 Claude Code 中说出以下关键词时，这个 Skill 会自动激活：

- "创建演示文稿"
- "生成 Excalidraw"
- "制作教学材料"
- "做个示意图"
- "画个架构图"

### 示例对话

```
User: 帮我创建一个关于 Docker 容器化的演示文稿，适合10分钟讲解

Claude: 好的，我会使用 excalidraw-presentation-generator skill
为你创建一个关于 Docker 的演示文稿...
```

## 📚 包含的最佳实践

这个 Skill 总结了以下最佳实践：

### 1. 视觉设计
- ✅ 统一的颜色系统
- ✅ 清晰的字体层次
- ✅ 合理的留白和间距
- ✅ 恰当使用 emoji

### 2. 内容组织
- ✅ 吸引人的开场
- ✅ 理论与实战平衡（40% + 60%）
- ✅ 左右分栏对比布局
- ✅ 流程图和箭头引导

### 3. 技术规范
- ✅ 完整的 JSON 结构
- ✅ 正确的元素字段
- ✅ 合适的坐标系统
- ✅ 一致的样式配置

## 🎨 模板说明

### basic-presentation.json
基础演示模板，包含：
- 标题和副标题
- 概念卡片
- 箭头过渡

适合：快速创建简单的演示文稿

### technical-tutorial.json
技术教程模板，包含：
- 代码展示区（灰色背景）
- 效果说明区（彩色背景）
- 左右分栏布局

适合：技术教学、代码示例展示

### comparison-layout.json
对比布局模板，包含：
- 推荐做法区（绿色）
- 避免做法区（红色）
- 提示说明

适合：最佳实践、避坑指南、对比分析

## 💡 使用建议

1. **明确目标**：确定演示时长和受众背景
2. **保持简洁**：每屏不超过3个主要概念
3. **注重实战**：理论讲解配合具体示例
4. **视觉引导**：使用箭头和颜色引导注意力
5. **测试效果**：在 Excalidraw 中打开验证效果

## 🎯 最佳实践清单

创建演示文稿前，问自己：

- [ ] 演示时长是多少？（决定信息密度）
- [ ] 听众是谁？（决定深度和术语使用）
- [ ] 核心信息是什么？（3-5个关键点）
- [ ] 需要哪些示例？（至少2-3个）
- [ ] 如何可视化？（流程图、对比、代码展示）
